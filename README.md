# Custom WordPress theme, by Gonzalo Stoll

To kickstart this theme stylesheet, run `sass --watch assets/scss/theme.scss:style.css`.


This theme has support for:

* Thumbnails

* Primary menu

* Custom post type

* jQuery

* Bootstrap

* Aos plugin (animate on scroll)

* Back to top animation

* Logo uploader

* Social media icons

* Fontawesome 5

* Easing

* Anchor menu

* LazyLoad
